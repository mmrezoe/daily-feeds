# daily feeds

first write config.yaml

### Installation

```
go install github.com/mmrezoe/daily-feeds@latest
```

or
git clone project then `go build .`

### First run

```sh
./daily-feeds -j
```

### Daily run

```sh
./daily-feeds
```
